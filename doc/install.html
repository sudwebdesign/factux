Installation de factux sur votre serveur


Version 5.0.0


Avant l'installation assurez-vous de recupérer

1. l'adresse de votre base de données

2. le mot de passe de l'utilisateur de votre base mysql

3. le login de votre utilisateur mysql

4. Si la base de données a été créée le nom de cette base de données vide

(si elle n'existe pas et que vous avez les droits nécessaires sur le serveur de bases de données, Factux la créera pour vous

5. Toutes les infos de votre entreprise (n° de tva, adresse, tel, registre de commerce...)



Décompressez le fichier dans un répertoire accessible par apache

Vérifiez les droits des fichiers (chmod)

Tous les fichiers doivent être lisibles par apache

Les répertoires

/include

/dump

/image

/fpdf

/include/session

et le répertoire racine de factux

doivent permettre un droit d'écriture par apache


Pointez votre navigateur sur http://votre_hebergeur.org/factux/installeur/

Suivez les instructions à l'écran.

Apres l'installation, il faut effacer le répertoire installer de l'arborescence de factux (si vous ne le faites pas un message vous le rappellera)

Réduire les droits des fichiers :

/include/config/var.php

/include/config/common.php

Ceux-ci ne doivent être accessibles qu'en lecture par apache.

Si vous ne le faites pas, un rappel vous sera adressé dans Factux


Bugs et desiderata sur http://factux.sourceforge.net

Upgrade de Factux


Faites tout d'abord une sauvegarde de votre base de donnée (en ligne de commande ou avec l'utilitaire de Factux ou encore via phpmyadmin ou tout autre soft de gestion de mysql)

Ensuite faites un backup des fichiers suivants :

/include/config/common.php

/include/config/var.php

Ceci fait, decompressez l'archive et uploadez les fichiers dans le répertoire de factux en ecrasant les anciens fichiers.

Verifiez les droits comme pour l'installation (voir plus haut)


Reuploader les fichiers

/include/config/common.php /

include/config/var.php

Si vous utliser l'euro comme devise:

Editez /include/config/var.php et remplacez et remplacer $devise ="&#128;"; par $devise ="&euro;";

# Si vous avez modifié les fichiers fact_pdf.php ou bon_pdf.php vous devrez malheureusement refaire ces modifications


!!!!!!!!!!!!! Je ne suis en aucun cas responsable des pertes de données dues à l'upgrade de Factux !!!!!!!!!!!!!


Ouvrez votre navigateur et aller http://votre hebergeur.com/factux/installeur/upgrade/

Suivez les instructions à l'ecran

Si vous avez modifié les fichiers fact_pdf.php ou bon_pdf.php vous devrez malheureusement refaire ces modifications
